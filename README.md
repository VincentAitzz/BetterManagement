## Descripción

Este proyecto es una aplicación que permite a los usuarios tener una mejor organizacion.

## Características

- **Gestión Personal**: Los usuarios pueden administrar su propio horario y añadir notas para eventos especiales en el calendario.
- **Gestión Empresarial**: Los administradores pueden registrar horarios de trabajo para sus empleados, visualizar todos los empleados, editar su información y gestionar las características de los usuarios promedio.

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/VincentAitzz/BetterManagment.git
   cd BetterManagment
   ```

## Uso
Para utilizar la aplicación, simplemente ejecútala en un dispositivo Android o en un emulador. A continuación, podrás crear tu perfil y comenzar a gestionar tus horarios y notas.
   ```bash
  ./gradlew build
   ```
